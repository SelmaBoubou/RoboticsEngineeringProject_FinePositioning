____________________________Creation of the package____________________________

- Open terminal
$cd ~/catkin_ws/src


- choose the name of the package and dependencies
$ catkin_create_pkg fine_positioning roscpp rospy urdf

 
$ cd fine_positioning 

- create the directory inside the package
$ mkdir maps
$ mkdir nodes
$ mkdir rviz
$ mkdir data
$ mkdir launch 

$ cd ~/catkin_ws

$ catkin_make

$ rospack profile
